Pemesanan Tiket Pesawat
Program tersebut adalah sebuah simulasi dalam pemesanan tiket pesawat berbasis teks yang memungkinkan pengguna untuk melihat daftar rute penerbangan beserta harganya, menghitung total biaya pembelian berdasarkan jumlah tiket yang telah dipesan, dan keluar dari sistem. Program ini dirancang dengan alur interaktif yang dimana program tidak hanya berjalan satu kali lalu selesai tetapi ia menunggu respon dan masukan dari pengguna yang kemudian menunggu intruksi selanjutnya,dan menggunakan looping dan penanganan kesalahan dasar untuk memastikan input yang dimasukan oleh pengguna sesuai dengan format yang diminta. Dalam struktur data program ini menerapkan List satu dimensi secara paralel untuk menyimpan data rute dan harga, dimana pada indeks daftar rute dan harga. algoritma yang digunakan adalah sekuensial atau percabangan yang dimana program mengeksekusi intruksi langkah demi langkah dengan menggunakan perulangan while untuk menjaga program tetap berjalan serta if, elif, else, untuk menentukan logika aksi berdasarkan pilihan menu yang diinputkan oleh pengguna.

Pada fungsi main(), program menyiapkan data menggunakan Struktur List. Rute digunakan untuk menyimpan teks nama tujuan dan harga untuk menyimpan angka nilai dari tiket tersebut. Kedua list ini bersifat paralel yang artinya data pertama di rute secara otomatis memiliki harga yang berada pada di urutan pertama dan seterusnya.

Pada running = true berfungsi sebagai variabel boolean yang berati menekan saklar untuk berjalan atau hidup. Ketika sampai ke while running, ia akan mengecek isi dari variabel tersebut. Karena isinya adalah true maka komputer memberikan kepada program untuk mengeksekusi kode di dalam perulangan  seperti menampilkan menu, dan menghitung harga. Pada try, untuk menjalankan baris dibawahnya tetapi jika terjadi kesalahan teknis maka jangan hentikan program tetapi melainkan lempar ke bagian penanganan atau except. Pada bagian except value error yang dimana jika kita memasukan angka yang tidak ada dipilhan maka muncul output pilihan tidak valid dan begitupun sebaliknya jika memasukan huruf. 
<img width="748" height="276" alt="image" src="https://github.com/user-attachments/assets/8d714d5e-d50c-48a3-9cdb-127c84fa9e48" />
Pada baris ini memilih angka 2 dan fungsi try berfungsi untuk menangkap kesalahan, yang dimana pada baris ini disuruh untuk memasukan pilih nomor rute. Kemudian  if 0 <= pilih < len(rute) digunakan untuk memastikan bahwa angka tersebut ada di dalam rute shingga ia akan melanjutkan ke proses perhitungan jumlah dan harga. Dimana ketika sudah memilih rute akan disuruh untuk memasukan harga tiket dari rute tersebut jika sudah maka komputer akan menghitung dan akan muncul output total yang harus dibayar. Else untuk ketika pengguna memasukan rute yang tidak ada dalam pilahan maka akan mncul output rute tidak ditemukan kemudian except ValueError digunakan untuk mendeteksi kesalahan yang dimana ketika pengguna memasukan input selain angka.

Pada baris ini memilih angka 3, program akan menjalankan baris berikutnya yaitu running = False berfungsi untuk mematikan saklar agar program berhenti berjalan sehingga akan muncul pesan program telah selesai. Sampai jumpa dan jika memasukan pilihan selain 1, 2, 3 maka akan muncul output pilihan tidak valid. if __name__ == "__main__" main() berfungsi untuk memastikan bahwa kode hanya dieksekusi saat file pyhton secara langsung.
